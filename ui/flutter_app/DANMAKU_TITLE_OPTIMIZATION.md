# 弹幕标题优化完成

## 问题
用户反馈：即使文件名是英文的，Emby 详情页也会显示中文标题（如"陈情令"），应该优先使用这个中文标题来匹配弹幕。

## 解决方案

### 1. 修改 Emby 页面 - 传递完整标题
**文件**: `ui/flutter_app/lib/emby_page.dart`

修改 `_playItem` 函数，从 Emby API 的 `item` 对象中提取：
- **中文标题**：`item['Name']`（如"陈情令"）
- **季数**：`item['ParentIndexNumber']`（如 1）
- **集数**：`item['IndexNumber']`（如 1）

构建完整标题传递给播放器：
```dart
// 电视剧集
陈情令 S01E01

// 电影
流浪地球
```

### 2. 修改弹幕控制器 - 智能匹配
**文件**: `ui/flutter_app/lib/features/danmaku/controllers/danmaku_controller.dart`

实现两种清理策略：

#### 策略 A：Emby 标题（已含季集信息）
如果标题已包含 S01E01 格式（来自 Emby）：
- 只移除技术信息（分辨率、编码等）
- 保留中文名称和季集信息
- 示例：`陈情令 S01E01` → `陈情令 S01E01`

#### 策略 B：文件名（需要提取中文）
如果标题不包含季集信息（来自文件名）：
- 优先提取中文名称
- 提取季集信息
- 移除所有技术信息
- 示例：`陈情令.The.Untamed.S01E01.2019.1080p.mkv` → `陈情令 S01E01`

## 匹配流程

### 电视剧集
```
1. Emby API 返回:
   - Name: "陈情令"
   - ParentIndexNumber: 1
   - IndexNumber: 1

2. 构建完整标题:
   "陈情令 S01E01"

3. 传递给播放器:
   widget.title = "陈情令 S01E01"

4. 弹幕匹配:
   - 检测到已含 S01E01
   - 使用简单清理（只移除技术信息）
   - 搜索: "陈情令 S01E01"

5. API 调用:
   GET /api/v2/match?fileName=陈情令 S01E01
```

### 电影
```
1. Emby API 返回:
   - Name: "流浪地球"
   - Type: "Movie"

2. 构建标题:
   "流浪地球"

3. 传递给播放器:
   widget.title = "流浪地球"

4. 弹幕匹配:
   - 没有季集信息
   - 使用完整清理逻辑
   - 搜索: "流浪地球"

5. API 调用:
   GET /api/v2/match?fileName=流浪地球
```

## 优势

### 1. 优先使用中文标题
- ✅ Emby 的 `Name` 字段通常是中文
- ✅ 即使文件名是英文，也能用中文匹配
- ✅ 提高匹配成功率

### 2. 准确的季集信息
- ✅ 从 Emby API 获取准确的季、集数
- ✅ 不依赖文件名格式
- ✅ 标准化为 S01E01 格式

### 3. 兼容性
- ✅ 兼容 Emby 标题（中文 + S01E01）
- ✅ 兼容文件名（英文 + 中文混合）
- ✅ 兼容电影（无季集信息）
