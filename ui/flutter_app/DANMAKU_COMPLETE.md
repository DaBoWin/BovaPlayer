## 使用方法

### 1. 播放视频
打开任何视频，播放器会自动尝试加载弹幕。

### 2. 开关弹幕
点击播放器底部的"弹幕"按钮即可开关弹幕显示。

### 3. 调整设置
长按"弹幕"按钮打开设置面板，可以调整：
- 透明度
- 字体大小
- 弹幕速度
- 显示区域

### 4. 查看弹幕信息
设置面板底部会显示当前视频的弹幕信息：
- 视频标题
- 弹幕数量

## 技术细节

### 弹幕匹配流程
1. 播放器初始化完成后自动调用 `_loadDanmaku()`
2. 使用视频标题调用 API 搜索匹配
3. 获取第一个匹配结果的弹幕数据
4. 按时间排序并加载到渲染引擎

### 弹幕渲染流程
1. 根据当前播放位置筛选应该显示的弹幕
2. 分配轨道（防止碰撞）
3. 计算弹幕位置和动画
4. 60fps 刷新渲染
5. 自动清理已经移出屏幕的弹幕

### 配置存储
弹幕设置保存在 SharedPreferences 中：
- `danmaku_enabled` - 是否启用
- `danmaku_opacity` - 透明度
- `danmaku_fontSize` - 字体大小
- `danmaku_speed` - 速度
- `danmaku_displayArea` - 显示区域

## 问题排查

### 弹幕不显示
1. 检查控制台日志，查看是否成功加载弹幕
2. 确认弹幕开关是否打开
3. 确认视频标题是否能匹配到弹幕

### 弹幕卡顿
1. 检查弹幕数量是否过多
2. 尝试降低弹幕密度（调整显示区域）
3. 检查设备性能

### API 调用失败
1. 检查网络连接
2. 查看控制台错误信息
3. 确认 API 服务是否正常
