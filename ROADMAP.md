# BovaPlayer 开发路线图

> 最后更新：2026-02-20

## 项目愿景

打造跨平台、高性能、智能聚合的现代媒体播放器，支持本地、NAS、媒体服务器、云盘等多种数据源，提供极致的播放体验。

## 当前状态（v0.2）

### ✅ 已完成
- 基础播放器（Media Kit + MPV）
- Emby 服务器集成
- 本地文件播放
- 播放位置记忆
- 基础 UI（Flutter）
- 字幕支持（SRT/ASS）
- 播放控制（进度、速度、画面比例）
- 网络播放优化（多线程、重连机制）

### 🔄 进行中
- 播放稳定性优化（TCP 超时、大文件缓冲）
- 网络性能调优

## 风险管理

### 技术风险
1. **硬件解码兼容性**
   - 缓解：设备能力检测 + 黑名单 + 软解降级

2. **HDR/DV 合规性**
   - 缓解：使用 libplacebo + 合规 Profile

3. **跨平台差异**
   - 缓解：统一抽象层 + 平台适配

4. **第三方依赖稳定性**
   - 缓解：版本锁定 + 备选方案

### 资源风险
1. **开发时间不足**
   - 缓解：优先级管理 + MVP 策略

2. **云服务成本**
   - 缓解：免费额度 + 成本监控

### 市场风险
1. **用户需求变化**
   - 缓解：快速迭代 + 用户反馈

2. **竞品压力**
   - 缓解：差异化功能 + 开放生态
