## 🎯 核心目标

### 1. 用户账号体系
建立三级账号体系，满足不同用户需求：

| 版本 | 定价 | 核心特性 |
|------|------|---------|
| **社区免费版** | ¥0 | 基础播放 + 有限同步（10 服务器，2 设备，100MB） |
| **Pro 版** | ¥68/年 或 ¥9/月 | 完整同步 + GitHub 同步 + 高级功能 |
| **永久版** | ¥298 一次性 | 终身 Pro 权益（无限服务器，无限设备，5GB 存储） |

### 2. 双模式云同步

#### 模式 1: Supabase 云同步（默认）
- ✅ 实时同步（< 2秒延迟）
- ✅ 开箱即用
- ✅ 跨设备快速
- ✅ 所有版本可用

#### 模式 2: GitHub 同步（Pro/永久版）
- ✅ 数据完全自主
- ✅ 隐私优先
- ✅ 无限存储
- ✅ Git 版本控制
- ✅ 免费无限

### 3. 数据同步范围

| 数据类型 | 同步方式 | 加密 | 备注 |
|---------|---------|------|------|
| **媒体服务器列表** | ✅ 云端 | ✅ Token 加密 | Emby/Jellyfin/Plex |
| **网络连接元数据** | ✅ 云端 | ❌ 不含密码 | SMB/FTP 连接信息 |
| **播放历史** | ✅ 云端 | ❌ 明文 | 位置、进度、时间 |
| **收藏列表** | ✅ 云端 | ❌ 明文 | 收藏的影片/剧集 |
| **用户设置** | ✅ 云端 | ❌ 明文 | 播放器设置、主题等 |
| **SMB/FTP 密码** | ❌ 仅本地 | ✅ AES-256 | 永不上传云端 |

## 🔒 安全策略

### 数据加密

| 数据类型 | 本地存储 | 云端存储 | 加密方式 |
|---------|---------|---------|---------|
| 用户密码 | ❌ | ✅ Hash | bcrypt |
| Emby Token | ✅ 加密 | ✅ 加密 | AES-256 |
| SMB 密码 | ✅ 加密 | ❌ 不上传 | AES-256 |
| FTP 密码 | ✅ 加密 | ❌ 不上传 | AES-256 |

### 安全原则
- 🔒 敏感密码永不上传云端
- 🔒 媒体服务器 Token 加密后上传
- 🔒 本地使用设备密钥加密
- 🔒 云端使用用户密钥加密
- 🔒 行级安全策略（RLS）
- 🔒 HTTPS 传输加密
