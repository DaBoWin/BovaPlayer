## 💡 注意事项

### 开发规范
1. 所有颜色使用 `DesignSystem.xxx`
2. 所有间距使用 `DesignSystem.spaceX`
3. 所有组件使用 `Bova` 前缀
4. 保持代码注释完整

### 测试要点
1. 每完成一个页面立即测试
2. 测试不同屏幕尺寸
3. 测试深色/浅色主题切换
4. 测试动画流畅度

### 性能优化
1. 使用 `const` 构造函数
2. 避免在 `build` 中创建新对象
3. 只动画 `transform` 和 `opacity`
4. 使用 `ListView.builder`
