# BovaPlayer v0.0.1 - 发布说明

## 🎉 首次发布

BovaPlayer 是一个跨平台的媒体播放器，支持本地播放和 Emby 媒体服务器。

## ✨ 主要功能

### 媒体播放
- 🎬 支持多种视频格式
- 🎵 音频播放支持
- ⏯️ 播放控制（播放/暂停/快进/快退）
- 🔊 音量控制
- 🔁 循环播放

### Emby 集成
- 📺 连接 Emby 媒体服务器
- 🏠 仪表板视图，显示媒体库
- 📁 浏览媒体库和文件夹
- 🎭 电影和电视剧支持
- 📺 电视剧按季/集组织
- 🖼️ 海报和背景图片显示
- 🔢 剧集数量徽章
- 🔍 递归搜索媒体内容

### 用户界面
- 🎨 现代化深色主题
- 🖱️ 悬停显示播放控制
- 📱 响应式布局
- 🔄 流畅的导航体验
- 🏷️ 媒体信息展示（标题、年份、评分等）

## 📦 支持平台

### ✅ macOS
- 原生应用程序包（.app）
- DMG 安装镜像
- 支持 macOS 10.13+

### ⚠️ Windows
- 需要在 Windows 上构建
- 或使用交叉编译（实验性）

### ⏳ Android
- 需要 Flutter 环境配置
- 需要添加 Android 平台支持

## 🚀 快速开始

### macOS
1. 下载 `BovaPlayer-macOS-v0.0.1.dmg`
2. 打开 DMG 文件
3. 将 BovaPlayer 拖到应用程序文件夹
4. 启动 BovaPlayer

### Windows
1. 下载 `BovaPlayer-Windows-v0.0.1.zip`
2. 解压到任意文件夹
3. 运行 `BovaPlayer.exe`

## 🔧 配置 Emby

1. 启动 BovaPlayer
2. 选择"媒体服务器"模式
3. 点击"添加服务器"
4. 输入服务器地址、用户名和密码
5. 连接并浏览媒体库

## 📝 已知问题

- Windows 版本需要在 Windows 系统上构建
- Android 版本需要完整的 Flutter 环境
- 某些视频格式可能需要额外的编解码器

## 🛠️ 技术栈

- **核心**: Rust
- **GUI**: egui/eframe
- **播放引擎**: MPV
- **网络**: reqwest
- **图片处理**: image
- **移动端**: Flutter (计划中)
